# Wishes
Wishes is a wishlist app written with TypeScript and NextJS using Firebase. More content coming soon.

## Planned Content
+ Create multiple lists
+ Others can view lists with or without an account
+ Others can mark items on others lists as bought with or without an account
+ Users can add friends to view their lists at any time
+ Auto-purchase tracking from external vendors
+ Lists auto-reset/remove already purchased after a specified date
+ Purchased items can be hidden from list owner to keep surprises
+ Add items from any website with automatic or manual addition of details

## Potential Post-Release Content
+ Browser extensions for adding items
+ Price comparison on items to find the best deal
+ List syncronization with Amazon or other websites
+ Mobile apps for quick access and sharing
